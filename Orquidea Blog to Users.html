Projeto Orquídea Inteligente

Monitoramento de umidade e cuidado automatizado para plantas sensíveis.

1. Visão Geral

O projeto de orquídea inteligente utiliza sensores e LEDs para monitorar a saúde e a umidade do substrato da planta, garantindo que ela receba o cuidado ideal. Este sistema é ideal para orquídeas em vasos com cascalho original.

2. Componentes Necessários

  • Sensor de Umidade MH séries
  • LEDs (Vermelho, Amarelo, Azul, Verde, Branco)
  • Resistores de 220Ω
  • Arduino Nano
  • Regulador LM7805
  • Bateria de 9V e conector
  • Protoboard e jumpers

3. Funcionamento

O sistema monitora os níveis de umidade e indica o estado do substrato com base em cores:

  • Vermelho: Solo muito seco, é necessário regar.
  • Amarelo: Solo levemente seco, atenção à próxima rega.
  • Azul: Umidade moderada, estado aceitável.
  • Verde: Umidade ideal, a planta está saudável.
  • Branco: Solo muito úmido, pare de regar.

4. Código Arduino

Este é o código que controla os LEDs com base nos sinais do sensor de umidade:

const int sensorPin = A0; const int leds[] = {2, 3, 4, 5, 6}; const int thresholds[] = {300, 500, 700, 900}; void setup() { for (int i = 0; i < 5; i++) { pinMode(leds[i], OUTPUT); } Serial.begin(9600); } void loop() { int moisture = analogRead(sensorPin); Serial.print("Umidade: "); Serial.println(moisture); for (int i = 0; i < 5; i++) { if (i == 4 || moisture < thresholds[i]) { digitalWrite(leds[i], HIGH); } else { digitalWrite(leds[i], LOW); } } delay(2000); }

5. Cuidados com a Orquídea

  • Mantenha o vaso em local iluminado, mas longe da luz solar direta.
  • Verifique os níveis de umidade regularmente.
  • Regue apenas quando os LEDs indicarem necessidade (vermelho ou amarelo).
  • Limpe o sensor de umidade com frequência para garantir leituras precisas.
